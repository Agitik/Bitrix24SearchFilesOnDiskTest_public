# Автотест поиска файлов на диске bitrix24

Тест проводит самую надежную проверку механизма поиска файлов на сайте битрикса.  
***
Для этого он формирует тестовую выборку:  
Открывает диск, заходит в каждую папку и выписывает название всех файлов и папок в документе.  

Все это реализовано на **Selenium**. Можно было проще. Но хотелось использователь только 1 установленный модуль python как для формирования выборки, так и для проведения тестирования непосредственно через графический интерфейс (Потому что на это можно еще накинуть скриншоты потом, да и имитация работы в браузере мне кажется надежнее, можно вживую увидеть, на чем упал тест)
  
Здесь уже интегрированы автоотчеты allure.   

Код предназначен для личного использования и опубликован только как пример реализации подобного автотеста.  
***
Как это использовать?  
1. Если нет сформированной тестовой выборки, то в файле **test_disk_search.py** нужно раскомментировать функицю **test_we_can_make_file_register(browser)**.
2. После этого запускать тест командой pytest с формированием json отчета для allure.
